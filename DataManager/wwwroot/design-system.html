Library Recommendations
EC
Apache ECharts
Chart library · via echarts-for-react
★ Recommended
The most feature-complete charting library for analytics. Powers ThinkingData, Alibaba Cloud, and major data platforms. Replaces C3.js completely.
  • 30+ chart types: line, bar, scatter, heatmap, funnel, sankey, treemap, gauge, calendar
  • First-class TypeScript types
  • Canvas renderer — handles 100k+ data points smoothly
  • Rich built-in tooltip, legend, zoom, brush interactions
  • LTV-specific: cohort heatmap, funnel drop-off, geo map
  • ~600KB bundle — use dynamic import for lazy loading
  • More verbose config vs Recharts
Use for: All production charts in ReportComponent. Drop-in C3.js replacement with vastly more power. The declarative option config maps cleanly to ReportDefinition.
RC
Recharts
Chart library · already in nova/
Alt / Simple
Pure-React declarative charts. Already used in the nova/ mockup. Good for simple KPI sparklines.
  • Pure React component model — very easy to integrate
  • Good for sparklines, simple line/bar in KPI cards
  • Smaller bundle when tree-shaken
  • Limited chart types (no heatmap, funnel, sankey)
  • Performance degrades above 5,000 data points
  • Custom rendering requires deep hacks
Keep for: Sparklines in KPI cards (very lightweight). Replace with ECharts for all main report charts.
TT
TanStack Table v8
Headless data grid · React
★ Recommended
The gold standard headless table for React. Gives you all the logic (sort, filter, paginate, group, column pin) while you own the rendering.
  • Headless — full design control; pairs perfectly with this design system
  • Virtual rows (TanStack Virtual) for 50k+ rows
  • Column pinning, row grouping, multi-sort
  • Excellent TypeScript — generics on row data type
  • 12k GitHub stars, actively maintained
  • More setup than a batteries-included grid
  • No built-in cell editors (need separate lib)
Use for: Cohort table, raw data table in report builder. Replaces the hand-rolled DataTable component in nova/ with proper sorting/filtering/pagination logic.
AG
AG Grid Community
Data grid · Enterprise-grade
Alt / Heavy
The most powerful grid available. Excel-like features, column groups, pivot, nested rows. Community edition is free.
  • Most feature-complete (pivot, tree data, Master-Detail)
  • Exceptional performance with virtual scrolling
  • Built-in cell editing, clipboard integration
  • ~1MB bundle — significant for a web app
  • Hard to override default styling; fights with custom design systems
  • Advanced features (row grouping, pivot) require Enterprise license
Consider only if: You need Excel-like in-cell editing or pivot tables in a future analytics workbench. For current use cases, TanStack Table is sufficient and lighter.
Recommended Stack Summary
Charts (main) Apache ECharts via echarts-for-react — drop C3.js entirely Charts (sparklines) Keep Recharts AreaChart in KPI cards — already in nova/ Data Grid TanStack Table v8 — headless, full design control, great TypeScript Virtual Scroll @tanstack/react-virtual — pair with TanStack Table for large datasets
